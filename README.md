# NThaCityRP Web System

A static multi-page website for the NThaCityRP Discord roleplay community hub.

## Description

This project is the official frontend portal for the NThaCityRP roleplay system. It provides a polished, dark street-level interface for civilian registration, police CAD, DMV licensing, business/faction coordination, applications, community support, and incident reporting.

## Features

- Police CAD dashboard with dispatch and warrant workflows
- Civilian registry system for RP identities
- DMV licensing and vehicle registration tools
- Business and faction management support
- Application hub for roles like PD, EMS, staff, and legal
- Donation/support page with compliant community language
- Complaint reporting system for incidents and officer review
- Discord integration with a join page and community channel preview

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Deployment Instructions

1. Ensure all files are committed to the repository.
2. Push the `main` branch to GitHub.
3. Deploy the site using GitHub Pages from the repository settings.

## GitHub Pages Instructions

1. Go to repo Settings → Pages
2. Set source to Deploy from branch
3. Select `main` branch
4. Select root folder `/`
5. Save
